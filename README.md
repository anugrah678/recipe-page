# Recipe Page

A modern and responsive recipe webpage built using HTML5 and CSS3, focusing on semantic markup, accessibility, and clean visual design.

## Features

* Hero image with optimized positioning
* Semantic HTML structure
* Accessible breadcrumb navigation
* Clear typography hierarchy
* Responsive layout
* CSS Grid for content organization
* Nutrition information displayed using semantic tables
* CSS custom properties for easy styling and maintenance

## What I Learned

Through this project, I strengthened my understanding of semantic HTML, accessibility best practices, typography systems, responsive design, CSS Grid, and maintainable CSS architecture. I also gained hands-on experience creating structured, user-friendly web pages that follow modern web development standards.
